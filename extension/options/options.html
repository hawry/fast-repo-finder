Configuration

Syntax

The configuration consist of a JSON array on the following format: 
  [ 
    { 
      "name": "string",
      "address": "string" 
    }
  ]
Where the attribute name is the text shown above the input field, and address is the target to open, with the query field marked as <frfterm>. The extension will do a simple replace of the <frfterm> keyword for whatever you put into the search field.

Example

  {
    "name": "PyPi",
    "address": "https://pypi.org/search/?q=<frfterm>"
  }
